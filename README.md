# 📊 Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn behavior using Power BI. The goal is to identify key factors that lead to customer churn and provide actionable insights to improve customer retention.

## 🛠️ Tools Used
- Power BI

## 📂 Dataset
The dataset contains customer information such as:
- Demographics (Age, Gender)
- Account details (Tenure, Contract type)
- Usage patterns
- Churn status

## 📈 Key Features
- Data cleaning and transformation in Power BI
- Creation of calculated measures and KPIs
- Interactive dashboards and visualizations

## 📊 KPIs Created
- Churn Rate
- Retention Rate
- Customer Lifetime Value (CLV)
- Monthly Charges Analysis

## 🔍 Insights
- Customers with shorter tenure have higher churn rates
- Month-to-month contracts show higher churn
- High monthly charges contribute to churn

## 📌 Dashboard Highlights
- Churn distribution by contract type
- Customer segmentation analysis
- Trend analysis of churn over time

## 🚀 Outcome
The dashboard helps stakeholders:
- Identify high-risk customers
- Improve retention strategies
- Make data-driven decisions

## 📁 Project Files
- Power BI (.pbix) file
- Dataset (CSV/Excel)
